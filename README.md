## Khởi động ứng dụng:

### Windows PowerShell:
```powershell
.\venv\Scripts\Activate.ps1
python app.py
```

### Sau đó mở trình duyệt:
http://localhost:5000

---

## Các lệnh thường dùng:

### Kích hoạt virtual environment:
```powershell
.\venv\Scripts\Activate.ps1
```

### Cài đặt thư viện (nếu cần):
```powershell
pip install flask
```

### Chạy ứng dụng:
```powershell
python app.py
```
